Step 1: Create the interface for the implementation class
	# mark class to @WebService (import javax.jws.WebService)
	# mark method to @WebMethod (import javax.jws.WebMethod)
	
Step 2: Mark the implementation class to @WebService and overwrite the method of interface

Step 3: Use wsgen ( Please go to the project bin dir before use wsgen)
	wsgen -verbose -cp . hello.HelloImpl -wsdl
	
Step 4: publish the web service and test it
	Endpoint endpoint = Endpoint.publish("http://localhost:9090/HelloWorld", new HelloImpl());
	System.out.println(endpoint.isPublished());

Step 5: Before code the client plz wsimport the webservice to local.
	wsimport -keep -d . -p hello.client http://localhost:9090/HelloWorld?wsdl
	
/******************From slide of COMP_6231_WebService ****************/
1. Code the implementation class.
2. Compile the implementation class.
3. Use wsgen to generate the artifacts required to deploy the service.
4. Package the files into a WAR file.
5. Deploy the WAR file. The tie classes (which are used to communicate with clients) are generated by the Application Server during deployment.
6. Code the client class.
7. Use wsimport to generate and compile the stub files.
8. Compile the client class.
9. Run the client.
